# genai-adoption-dashboard
Genai-adoption-dashboard -2026
 Project Overview

This dashboard analyzes how enterprises across 5 industries and 6 countries are adopting GenAI tools (ChatGPT, Claude, Copilot, Gemini) in 2026. It surfaces key patterns in adoption rates, productivity gains, revenue impact, and employee satisfaction — told as a data story, not just a chart dump.

Built as a portfolio project to demonstrate end-to-end data analytics skills: data exploration, aggregation, insight generation, and interactive visualization.
